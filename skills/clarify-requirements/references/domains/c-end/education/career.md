## 匹配信号
- 关键词：考证 / 职业 / IT / 编程 / 职场 / Excel / PMP / 考公 / 技能提升 / 转行 / 求职。
- 场景：下班学编程、备考 PMP / 考公、刷面试题求职。

## 继承要点（速览）
教育的学练测评闭环适用；职业技能的增量是——**项目实操 + 可验证产出（证书 / 就业）**。

## 核心价值要素
帮职场人 / 求职者**掌握一项可变现的技能或拿证**。不可缺：① **成体系的项目 / 案例实操**（不只看视频）② **可验证的产出 / 认证**（项目 / 证书 / 题测）。

## 业界标配功能
- 体系化**学习路径**（入门 → 进阶 → 实战，按目标 / 基础推荐）。
- **实操环境**：编程（在线 IDE / 沙箱）、设计（作业批改）、办公（实操作业）。
- 题库 / 模考（考证类）。
- 产出导向：简历项目 / 面试题 / 作品集。
- 社区 / 答疑 / 同伴 / 导师。
- 证书 / 学时 / 学习记录。

## 业界标杆做法
- **极客时间 / 慕课网**：IT 体系课。
- **Coursera / Udemy**：通识 + 证书。
- **牛客 / LeetCode**：刷题求职闭环。
- **三节课 / 开课吧**：职业技能。

## 常见陷阱
- **只播视频无实操** → 看了 ≠ 会。
- 学习路径过长 → 劝退。
- 无反馈 / 批改 → 自学式，完课率低。
- 证书不被市场认可 → 无变现价值。
- 忽视"**学完能做什么**"的就业衔接。

## 本节点专属澄清
- 哪类**职业技能 / 考证**？
- 要不要**实操环境**（在线编码 / 作业批改）？
- 如何验证"**学会了**"（项目 / 证书 / 题测）？
- 有无**就业 / 求职**衔接（简历项目 / 面试题）？
