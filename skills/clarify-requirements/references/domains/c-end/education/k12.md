## 匹配信号
- 关键词：中小学 / 学科（语数英物化生政史地）/ 课本 / 课纲 / 单元 / 期中期末 / 中考高考 / 家长 / 错题本 / 作业 / 搜题。
- 场景：放学后用平板做数学题、考前冲刺复习、家长查学情。

## 继承要点（速览）
教育的学练测评闭环全适用；K12 的增量是——**紧贴课纲 + 家长角色 + 未成年人合规**。

## 核心价值要素
帮中小学生把**校内学科**学会、考好。不可缺：① **紧贴教材 / 课纲**的知识点结构（学校不认的内容没价值）② 练习 → 批改 → 错题 → 重练闭环。

## 业界标配功能
- 按**教材版本 / 年级 / 学期 / 单元**组织（人教、北师大、苏教…）。
- 题型覆盖（选择 / 填空 / 解答 / 实验）+ **拍照搜题 / 智能批改**。
- **错题本**：自动收录 + 重练 + 举一反三。
- 知识点掌握度与薄弱定位。
- 同步校内进度 / 作业。
- **家长端**：学情报告 / 时长管控 / 监督（K12 强需求）。
- 考前冲刺 / 真题 / 模考。

## 业界标杆做法
- **作业帮 / 小猿搜题**：拍照搜题 + 解析（注意防"抄答案"）。
- **洋葱学院 / 学而思**：动画讲解 + 严格对齐课纲。
- **百词斩 / 扇贝**：小学英语词汇 SRS。

## 常见陷阱与反模式
- 搜题变"抄答案" → 反害学习（必须有解析 / 举一反三 / 限时）。
- 内容不贴课纲 → 学校和家长不认。
- 家长端过重 → 干扰孩子、引发抵触。
- 纯题海无讲解 → 只练不学。
- **忽视护眼 / 时长 / 内容审核** → K12 的合规与口碑红线。

## 本节点专属澄清
- 覆盖哪些**学科 / 年级 / 教材版本**？
- 是否**同步校内**进度 / 作业？
- 有没有**家长端**？家长要看什么、管什么？
- 效果怎么衡量（提分 / 掌握度 / 错题清零）？
- 合规要求（未成年人保护、时长限制、内容审核）？
