## 匹配信号
- 关键词：英语 / 日语 / 韩语 / 口语 / 听力 / 词汇 / 单词 / 跟读 / 发音评分 / 备考雅思托福四六级 / 语伴 / AI 对话。
- 场景：通勤背单词、跟读练口语、考前刷题。

## 继承要点（速览）
教育的学练测评闭环适用；语言学习的增量是——**输出练习（听说）+ 发音即时反馈**。

## 核心价值要素
提升某门语言的**听说读写**能力。不可缺：① 可练（跟读 / 造句 / 做题）② **即时反馈**（发音评分 / 对错判定）。

## 业界标配功能
- 词汇：SRS 背单词 + 例句 / 发音。
- **听说**：跟读 + 发音评分 + 对话练习 / AI 语伴。
- 读写：阅读 / 语法 / 写作批改。
- **备考定向**：四六级 / 雅思 / 托福 / JLPT 等题型与机经。
- 情景 / 主题分类（旅行 / 商务 / 日常）。
- 打卡 / 连续 / 排行榜。

## 业界标杆做法
- **Duolingo**：游戏化句型 + streak。
- **流利说 / Elsa Speak**：发音评分 + 音素级纠错 + AI 口语。
- **百词斩 / 扇贝 / 墨墨**：词汇 SRS。

## 常见陷阱
- 只背词不练用 → "哑巴英语"。
- **发音反馈太粗**（只到句 / 词级，到不了音素）→ 改不了发音。
- 只有输入无输出（不张嘴）→ 学不会说。
- 游戏化过度 → 稀释学习密度。
- 难度不自适应 → 太简单无聊 / 太难放弃。

## 本节点专属澄清
- 哪**门语言**？
- 侧重**听说读写**哪个？
- 是否**备考某考试**（决定题型与机经）？
- 发音反馈要多细（句 / 词 / **音素**级）？
- 有没有**输出 / 张嘴**练习？
