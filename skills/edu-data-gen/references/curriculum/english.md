# 英语 课纲与知识点（内置库）

> subject code: `en`。G9 课纲对齐门事实源之一（`source=builtin`）。种子库，可扩展。
> id 规约：`kp-en-<grade>-<slug>`。年级：g3…g9。

## 小学（g3-g6）—— 主题词汇 + 基础句型

小学英语以**主题**为单元组织词汇与句型（对齐词汇课主题式大纲）。

### g3
- `kp-en-g3-greetings` 问候与自我介绍 — Hello/Goodbye/I'm...
- `kp-en-g3-colors` 颜色 — red/blue/What color is it?
- `kp-en-g3-numbers-1-20` 数字 1-20 — How many?

### g4
- `kp-en-g4-family` 家庭成员 — father/mother/parents/This is my...
  - 考点：家庭成员词、介绍句型
- `kp-en-g4-animals` 动物 — cat/dog/elephant/Do you like...?
- `kp-en-g4-food` 食物 — apple/rice/What do you like?

### g5
- `kp-en-g5-family` 家庭（扩展） — parents/grandparents/uncle
  - 前置：`kp-en-g4-family`
- `kp-en-g5-daily-routine` 日常作息 — get up/go to school/一般现在时
- `kp-en-g5-weather` 天气 — sunny/rainy/How's the weather?

### g6
- `kp-en-g6-hobbies` 爱好 — reading/swimming/like doing
- `kp-en-g6-past-events` 过去的事 — went/did/一般过去时初步
  - 前置：`kp-en-g5-daily-routine`

## 初中（g7-g9）—— 语法体系 + 话题阅读

### g7
- `kp-en-g7-present-simple` 一般现在时 — 第三人称单数、频率副词
- `kp-en-g7-present-continuous` 现在进行时 — be + doing
- `kp-en-g7-comparatives` 形容词比较级 — than

### g8
- `kp-en-g8-past-simple` 一般过去时 — 规则/不规则动词、did
  - 前置：`kp-en-g6-past-events`
- `kp-en-g8-future-going-to` be going to / will
- `kp-en-g8-clause-if` 条件状语从句（if 引导）

### g9
- `kp-en-g9-present-perfect` 现在完成时 — have/has done、already/yet
  - 前置：`kp-en-g8-past-simple`
- `kp-en-g9-passive-voice` 被动语态 — be + done
- `kp-en-g9-reading-inference` 阅读推理 — 推断作者意图/词义
