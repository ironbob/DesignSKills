# 科学 课纲与知识点（内置库）

> subject code: `science`。G9 课纲对齐门事实源之一（`source=builtin`）。种子库，可扩展。
> id 规约：`kp-science-<grade>-<slug>`。年级：g3…g9。
> 说明：小学为"科学"综合科；初中科学含物质科学（物理基础）、生命科学（生物基础）、地球与宇宙，可按主题拆分知识点。

## 小学科学（g3-g6）—— 物质 / 生命 / 地球宇宙

### g3
- `kp-science-g3-plant-parts` 植物的身体 — 根茎叶花果实
- `kp-science-g3-magnets` 磁铁 — 磁极、磁力
- `kp-science-g3-states-of-matter` 物质状态 — 固液气初步

### g4
- `kp-science-g4-sound` 声音 — 振动产生、传播
- `kp-science-g4-circuit-simple` 简单电路 — 通路、开关
  - 前置：`kp-science-g3-magnets`
- `kp-science-g4-rock-mineral` 岩石与矿物 — 常见岩石

### g5
- `kp-science-g5-light` 光 — 直线传播、反射、折射
- `kp-science-g5-ecosystem` 生态系统 — 食物链、栖息地
- `kp-science-g5-earth-sun-moon` 地球与日月 — 昼夜、月相

### g6
- `kp-science-g6-energy` 能量及其转化 — 动能/势能/热能
- `kp-science-g6-human-body` 人体系统 — 循环/呼吸/消化
- `kp-science-g6-weather` 天气与气候 — 气象要素

## 初中科学（g7-g9）—— 物理/化学/生物基础

### g7
- `kp-science-g7-measurement` 科学测量 — 长度/质量/时间/误差
- `kp-science-g7-cell` 细胞 — 结构、动植物细胞
- `kp-science-g7-motion-basic` 机械运动 — 速度、参照物

### g8
- `kp-science-g8-force` 力 — 重力/摩擦力/牛顿第一定律
  - 前置：`kp-science-g7-motion-basic`
- `kp-science-g8-atom-molecule` 原子与分子 — 分子动理论
- `kp-science-g8-chemical-change` 物理变化与化学变化 — 判别
  - 前置：`kp-science-g3-states-of-matter`

### g9
- `kp-science-g9-electricity` 电学基础 — 欧姆定律、串并联
  - 前置：`kp-science-g4-circuit-simple`、`kp-science-g8-force`
- `kp-science-g9-chemical-equation` 化学方程式 — 质量守恒、配平
  - 前置：`kp-science-g8-chemical-change`
- `kp-science-g9-genetics` 遗传与变异 — 基因、性状
  - 前置：`kp-science-g7-cell`
